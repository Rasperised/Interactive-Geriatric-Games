## M5StickC Combined Controller (For Demo)

### Description
This folder contains a combined M5StickC controller firmware prepared
specifically for the final demo presentation.

Only selected control modes are enabled to simplify setup and mode switching
during live demonstration.

---

### Mode Switching
- Press **Button A** on the M5StickC to switch between enabled demo modes

---

### Display
- The M5StickC LCD displays the currently active demo mode

---

### Notes
- Scripts are written for the **Arduino IDE**
- Demo-focused configuration with reduced modes
- Used during the final presentation and live demonstration
