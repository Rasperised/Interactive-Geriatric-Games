## Tiles

Tiles is a picture-based puzzle game where players reconstruct an image by rearranging shuffled tiles into their correct positions. The game focuses on visual recognition and spatial reasoning through simple, direct interaction.

This game is part of a larger interactive games platform developed for geriatric engagement and rehabilitation.

---

### Interactive Hardware
- Touch Screen Monitor

---

### Gameplay Overview
- A complete image is divided into multiple tiles and shuffled
- Players rearrange the tiles to reconstruct the original image
- The game is completed once all tiles are placed correctly

---

### Therapeutic Intent
- Supports visual recognition and spatial awareness
- Encourages attention and problem-solving through image reconstruction
- Promotes calm, self-paced interaction without time pressure
- Designed to encourage repeated engagement through familiar imagery

---

### Key Features
- Allows users to upload their own image and play the game using that image
- Supports capturing a photo on the spot and using it immediately for gameplay
- Includes pre-loaded images available for selection within the game
- Two difficulty modes:
  - Easy: 8 tiles
  - Medium: 12 tiles

---

### Controls
- **Touch Input:** Touch and drag to interact with and move tiles
- **Computer Mouse:** Click and drag tiles using the left mouse button

---

### Technical Details
- Engine: Unity
- Unity Version: 6000.2.7f2 (Unity 6 – Tech Stream)
- Platform: PC
- Input: Mouse / Touch

---

### How to Run
1. Download the repository or use **Code → Download ZIP**
2. Open **Unity Hub**
3. Add this folder (`unity/Tiles`) as a project
4. Open the **Scenes** folder and load the main scene into the Hierarchy
5. Press Play in the Unity Editor

---

### Notes
- The `Library` folder is excluded from version control and will be regenerated automatically by Unity on first launch.
