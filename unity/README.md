# Unity Games Folder

This folder contains all Unity projects developed for the Interactive Geriatric Games FYP.

Each subfolder is a standalone Unity project and can be opened independently in Unity Hub.

---

## Projects

- MemoryCard
- Tiles
- FlappyBird
- Pong
- FlowerField
- TargetBall

---

## Opening a Project

1. Open Unity Hub
2. Select **Add → Add project from disk**
3. Choose one project folder (e.g. `unity/MemoryCard`)
4. Open the project and press Play

---

## Notes

- `Library` folders are intentionally excluded and will be regenerated by Unity.
- Some projects use external inputs (e.g. sensors or OpenCV). Refer to each project’s own README for details.
- Developed and tested using **Unity 6 (6000.2.7f2)**.
